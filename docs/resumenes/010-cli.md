# RADAR-010: adapters/cli.py — CLI

Expone el engine por línea de comandos con tres subcomandos: `radar index <repo>` para construir y persistir el índice, `radar query "<code>"` para buscar funciones similares e imprimir los resultados formateados, y `radar serve` para levantar el MCP server. Sirve como herramienta de desarrollo y como fallback de demo si el MCP no conecta a tiempo.
